# Layout Group

## Definition

**Layout Group** is a named collection of [Layouts](../layout/) that can be applied to a [Display](../display/).

Each Layout Group has a name and contains multiple [Layouts](../layout/) (e.g., "left half", "right third", "full screen").

## Examples

- Layout Group "Development" with Layouts: left half, right half, full screen
- Layout Group "Presentation" with Layouts: full screen, center 80%

## Collocations

- apply (a Layout Group to a Display)
- select (a Layout Group)

## Related Concepts

- See [Layout](../layout/) for the individual window arrangements in a Layout Group
- See [Display](../display/) for where Layout Groups are applied
- See [Space](../space/) for the container that holds multiple Displays
