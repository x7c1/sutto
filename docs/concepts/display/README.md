# Display

## Definition

**Display** represents a physical monitor and the [Layout Group](../layout-group/) applied to it.

Each Display identifies:
- Which physical monitor it refers to
- Which [Layout Group](../layout-group/) is currently applied to that monitor

## Examples

- Display for Monitor 0 with "vertical 3-split" Layout Group applied
- Display for Monitor 1 with "horizontal 2-split" Layout Group applied

## Collocations

- apply (a Layout Group to a Display)

- See [Space](../space/) for the container that holds multiple Displays
- See [Layout Group](../layout-group/) for the layouts applied to a Display
