# License Key

## Definition

**License Key** is a unique string identifier that a user receives upon purchasing a subscription. It is used to activate a [License](../).

A License Key exists from the moment of purchase, while a [License](../) is created only when the key is activated on a device.

## Examples

- A user receives a License Key in their purchase confirmation email
- A user enters a License Key in the preferences to activate a License

## Collocations

- enter (a License Key) — input the key in the preferences
- activate (a License Key) — use the key to create a License on the device

## Related Concepts

- See [License](../) for the authorization created when a License Key is activated
