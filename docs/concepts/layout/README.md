# Layout

## Definition

**Layout** is a window arrangement that defines where and how a window should be placed on the screen.

Each Layout specifies:
- [Layout Position](./layout-position/) (where the window appears)
- [Layout Size](./layout-size/) (how large the window is)

## Examples

- "Left half" - window occupies the left 50% of the screen
- "Right third" - window occupies the right 1/3 of the screen
- "Full screen" - window occupies the entire screen
- "Top-left quarter" - window occupies the top-left 25% of the screen

## Collocations

- select (a Layout)
- apply (a Layout to a window)

## Related Concepts

- See [Layout Group](../layout-group/) for the collection that contains multiple Layouts
