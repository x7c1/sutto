## Directory Structure

| Directory | Description |
|---|---|
| `concepts/` | Living documentation defining shared product vocabulary |
| `examples/` | Example configuration files (e.g., monitor layouts) |
| `guides/` | How-to guides for development |
| `learning/` | Learning notes and reference material |
| `plans/` | Planning documents for past and upcoming work |
| `proposals/` | Improvement proposals for review and triage |

See [Documentation Structure](guides/10-documentation-structure.md) for details.
