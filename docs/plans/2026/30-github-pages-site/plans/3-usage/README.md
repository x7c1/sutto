# Sub-Plan 3: Usage Page

## Overview

Create a user-facing usage guide explaining how to use Snappa's core features.

## Content

- **Drag-to-Snap**: How to drag windows to screen edges to trigger the layout panel
- **Main Panel interaction**: Selecting a layout, keyboard navigation (arrow keys, Enter, Escape)
- **Keyboard shortcut**: How to configure and use the optional keyboard shortcut
- **Layout customization**: Brief overview of the Spaces page in preferences

## Verification

- Page renders correctly in local preview
- Instructions are accurate and easy to follow for non-technical users
