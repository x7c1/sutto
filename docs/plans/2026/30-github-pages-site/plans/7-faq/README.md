# Sub-Plan 7: FAQ Page

Status: Draft

## Overview

Create a frequently asked questions page covering common user questions.

## Content

- **What counts as a trial day?** — Actual usage days, not calendar days. A day is counted when the extension is first used on that day.
- **Can I use Snappa on multiple devices?** — Yes, up to 3 devices per license. Activating on a 4th device deactivates the oldest.
- **What happens when my trial expires?** — The extension is disabled until a license is purchased.
- **Do I need internet to use Snappa?** — Internet is needed for license validation, but the extension works offline for up to 7 days after the last successful validation.
- **Which GNOME Shell versions are supported?** — GNOME Shell 46.
- **How do I report a bug?** — Link to GitHub Issues.
- **How do I set a keyboard shortcut?** — Link to Usage page.

## Verification

- Page renders correctly in local preview
- Answers are accurate and consistent with extension behavior
