# Pricing

## Free Trial

Sutto includes a **30-day free trial** with full access to all features. Trial days are counted based on actual usage — only days when you use the extension count toward the 30-day limit, not calendar days.

No credit card is required to start the trial.

## Subscription

After the trial, a subscription unlocks continued access to all features:

- All features included (edge snapping, keyboard shortcuts, layout customization)
- Up to **3 device activations** per license
- Ongoing updates and improvements

::: tip Ready to subscribe?
[Purchase a License](https://sutto.example.com/purchase) to continue using Sutto after your trial ends.
:::

## Trial vs. Licensed

| | Free Trial | Licensed |
|---|---|---|
| All features | Yes | Yes |
| Duration | 30 usage days | Subscription period |
| Device activations | — | Up to 3 |
